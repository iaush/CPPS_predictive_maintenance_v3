Project Information

The current data driven equipment maintenance approach lacks of recommended maintenance solution. This project proposes prescriptive maintenance to determine the root cause of the impending failure and provides the best maintenance action to optimize the total cost.

Objective

System level prescriptive maintenance

System level prescriptive maintenance algorithm predicts degradation level while also determining the root cause of the system failure to facilitate the decision of the maintenance actions.

Offline deep reinforcement learning

Offline deep reinforcement learning algorithm to recommend maintenance actions to optimize the total cost.
