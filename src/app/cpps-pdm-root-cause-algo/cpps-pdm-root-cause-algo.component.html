WP 3.3 Maintenance Root Cause Algorithm (Algorithm 1)

CPPS WP 3.3 Algorithm:

Proposed a new framework which integrates with domain expert knowledge with feature anomaly detection methods to best identify/classify root cause of maintenance. The F1 score for CPPS WP 3.3 algorithm is higher than the state-of-the-art algorithms.
